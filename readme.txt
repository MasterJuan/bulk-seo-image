Easily update all the ALT attribute of your images. 
Can add the Article Name automatically to all your images improving traffic from search engines.

== Description ==
Image ALT attribute is certainly significant for SEO and usability. 
Increase the Google Search Visibility of your articles with BULK SEO Image
Without BULK SEO Image, you need to write manually the ALT attribute of each image, or you can use a plugin that does it for you, but it may slow down the loading speed of your page
Now, with BULK SEO Image, this light and simpe plugin, you can update all the ALT attributes at once.
Save one at a time, or save all changes with one click

What about the next versions ? :
Generate Google Sitemap for Image
More options for the new ALT attribute value name
...

== Installation ==
1. Upload the whole plugin folder to your /wp-content/plugins/ folder.
2. Go to the Plugins page and activate the plugin.
3. Use the Options page, BULK SEO Image, and modify the ALT attribute in series
